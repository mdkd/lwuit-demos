LWUIT Birthdays
=========================

LWUIT Birthdays is an example project demonstrating how to use the LWUIT
Calendar component together with the Series 40 PIM APIs to create a simple
Birthdays application.

5. Building, installing, and running the application
-------------------------------------------------------------------------------

5.1 Preparations
----------------

Before opening the project, make sure Nokia SDK 2.0 for Java is installed and 
added to NetBeans.

5.2 Building
------------

The project can be easily opened in NetBeans by selecting 'Open Project' 
from the File menu and selecting the application. Building is done by
selecting 'Build main project'.

5.3 Series 40 device
--------------------

You can istall the application on a phone by transferring the JAR file 
via Nokia Suite or over Bluetooth.
